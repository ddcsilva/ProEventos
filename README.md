# ProEventosApp

Este repositório contém o código-fonte da aplicação ProEventos, desenvolvida com Angular e .NET 6 durante o curso da Udemy: [Angular, .NET Core e EF Core](https://www.udemy.com/course/angular-dotnetcore-efcore/).

## Tecnologias utilizadas

- Angular
- .NET 6

## Instalação e Execução

1. Clone o repositório: `git clone https://github.com/<seu_usuario>/ProEventosApp.git`

2. Navegue até a pasta do projeto: `cd ProEventosApp`

3. Instale as dependências do projeto:

- Front-end (Angular):
`cd ProEventos-SPA`
`npm install`

- Back-end (.NET 6):
`cd ProEventos-API`
`dotnet restore`

4. Execute o projeto:

- Front-end (Angular):
`ng serve`

- Back-end (.NET 6):
`dotnet run`

5. Acesse a aplicação em seu navegador:

- Front-end (Angular): http://localhost:4200/
- Back-end (.NET 6): http://localhost:5000/
